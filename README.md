# **Raíz Viva – Página Web E-Commerce de Plantas**

## **Objetivo**

Raíz Viva es una página web de comercio electrónico especializada en la venta de plantas de interior, exterior y productos relacionados con jardinería.
El proyecto tiene como objetivo proporcionar una plataforma moderna, rápida y accesible para que los usuarios puedan:

- Explorar el catálogo de productos.
- Agregar artículos al carrito.
- Crear una cuenta y gestionar su información.
- Realizar compras de forma sencilla e intuitiva.
- Administrar productos, ventas y clientes desde un panel para administradores.

## **Instrucciones de uso**

1. Requisitos

- XAMPP / WAMP / Laragon (PHP 8+ y MySQL).
- Navegador actualizado.
- MySQL Workbench (opcional para el manejo de BD).

2. Ejecución local

- Clona o descarga el repositorio.
- Coloca la carpeta del proyecto dentro de htdocs/.
- Importa la base de datos ubicada en /database/ecommerce_db.sql.
- Configura la conexión en:
    app/config/connectionController.php
  con tus credenciales de MySQL.
- Abre el proyecto desde el navegador:
    http://localhost/raizviva/
